# Smart AI Hiring System

## Overview

Smart AI Hiring System is a Flask-based dashboard for ranking candidates
using TF-IDF and cosine similarity.

## Features

-   Upload `.txt` resumes
-   AI-based candidate ranking
-   Leaderboard view
-   Excel export
-   Interactive dashboard
-   Google Colab compatible

## Main Technologies

-   Python
-   Flask
-   Pandas
-   NumPy
-   Scikit-Learn
-   OpenPyXL
-   Chart.js

## Project Structure

Single Google Colab cell containing: 1. Candidate pool generation 2.
Flask server 3. HTML/CSS/JavaScript UI 4. Resume upload API 5. Candidate
ranking API 6. Excel export 7. Colab iframe launch

## Running in Google Colab

1.  Execute the cell.
2.  Wait for the Flask server to start.
3.  Open the generated proxy URL.
4.  Upload resume files.
5.  Enter a job description.
6.  View ranked candidates.
7.  Export results to Excel.

## Output

-   Web dashboard
-   Candidate leaderboard
-   Ranked scores
-   Excel report
